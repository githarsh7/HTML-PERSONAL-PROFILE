# Personal Profile Webpage

This project is a simple **personal profile webpage** built using **semantic HTML**. It showcases my basic information, skills, and contact details in a clean and structured format.

## Features

- Semantic HTML structure  
- Clean and minimal design  
- Sections for About, Skills, and Contact  
- Beginner-friendly project  

## Technologies Used

- HTML5  

## Project Structure

- `index.html` – Main webpage file  

## Getting Started

To view the project locally:

1. Download or clone the repository  
2. Open `index.html` in your browser
